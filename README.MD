# Proyecto: Aplicación Web de Ajedrez Multijugador

El proyecto consiste en una **aplicación web de ajedrez multijugador** que permite a los usuarios jugar partidas en **tiempo real**, tanto contra otros jugadores como contra un **bot con inteligencia artificial ajustable**.

## Funcionalidades principales

- Sistema de **registro y login** con **confirmación por email**.
- Partidas multijugador en tiempo real.
- Modo de juego contra **IA con dificultad configurable**.
- **Sistema de puntos** basado en el rendimiento de cada partida.
- **Progresión por niveles** (10 niveles en total).

## Sistema de niveles

Los usuarios ascienden de nivel según su desempeño, organizados en las siguientes categorías:

- **Principiante**: niveles 1 a 3  
- **Amateur**: niveles 4 a 6  
- **Experto**: niveles 7 a 9  
- **Senior Player**: nivel 10  

## Estadísticas y experiencia de usuario

- Panel de **estadísticas personales**.
- **Historial de partidas** detallado.
- Diseño **responsive**, adaptable a distintos dispositivos.
- **Efectos interactivos** para mejorar la experiencia de usuario.

## Arquitectura y tecnologías

El desarrollo seguirá una **arquitectura estructurada**, utilizando las siguientes tecnologías:

- PHP  
- JavaScript  
- HTML  
- CSS  
- Bootstrap  
- Docker  

## Enfoque y sostenibilidad

El proyecto se plantea como una **idea de negocio sostenible**, orientada a:

- Promover el **razonamiento lógico**.
- Fomentar la **inclusión digital**.
- Ofrecer una plataforma accesible y educativa basada en el ajedrez.
